Design notes
→
Each bubble wraps the MoodFace SVG sticker (same paths as RewardStickers.tsx)
→
Bubble size = mood count: 3 occurrences → 108px, 2 → 84px, 1 → 60px
→
Fill colors from MOOD_FACE_COLORS in lib/moodFace.ts (happy=#FBEA9E, calm=#BFE0C0, etc.)
→
Light: face uses ink #141313 stroke on pastel fill. Dark: face stroke = fill color (no harsh ink on dark bg)
→
Soap bubble effect: radial-gradient + ::after shine crescent at 38% 30%
→
Chips: same MoodFace at 14px + mood label + count — uses color-mix tint pattern
→
Summary bar reuses dominant mood face at 22px
